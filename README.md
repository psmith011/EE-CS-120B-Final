EE-CS-120B-Final
================

My final project for EE/CS120B, including source code as well as associated header files

This project is a game called UFO hunter:
It uses:
1x ATMEGA32
5x Shift registers
2x 8x8 RGB LED MAtrix
1x 16x1 LC Display
4x push buttons
2x buzzers

The game is a first person shooter.
